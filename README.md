# **Course Overview**

**Humanitarians AI — Algorithms & Data Structures (Grace 6205 Edition)** is a community-built, open-access course designed to introduce learners to foundational and advanced algorithmic techniques.
This course blends:

* Classic data structures
* Algorithm analysis
* Graphs and greedy strategies
* Dynamic programming
* Network flow
* Approximation and randomized algorithms
* Algorithmic thinking & formal reasoning
* Python implementations for real-world problem solving

Volunteers “learn by teaching” by building labs, walkthroughs, visualizations, and short-form YouTube lessons.

This course includes:

* Hands-on Python algorithm implementations
* Step-by-step proofs and runtime analysis
* Algorithm visualization exercises
* Problem-solving sessions with Grace 6205
* Optional advanced modules for data engineering & AI careers

---

# **Course Structure & Pedagogy**

This course uses the Humanitarians AI **Learn → Build → Teach** framework:

1. **Start with an Outline**
   Volunteers help expand module plans and lab structures.

2. **Build Labs First**
   Each topic includes:

   * Python implementations
   * Small assignments
   * Visualizers or notebooks

3. **Use Grace 6205 (Algorithms Tutor GPT)**
   Volunteers generate:

   * Lecture scripts
   * Code explanations
   * Step-by-step proofs
   * Visual backfills

4. **Record YouTube Lectures**
   Short 5–12 minute focused lessons on each major concept.

5. **Continuous Updates**
   Community contributions through GitHub pull requests.
